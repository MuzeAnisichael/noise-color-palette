# 噪声调色盘

[简体中文](README.md) | [English](README.en.md)

一个无依赖的浏览器噪声合成与混音工具。它把“噪声颜色”做成可拖动的色盘，同时提供独立的混音器，用于调整 EQ、混响、声像、空间宽度和导出音频。

## 功能

- 色盘拖动实时控制噪声颜色。
- 中心接近白/灰噪声，越靠外频谱染色越强。
- 内置棕/红、粉红、黄色中低频、绿色中频、青色存在感、蓝色明亮、紫色空气感锚点。
- 中英文界面切换，并通过 `localStorage` 记住语言。
- 实时频谱曲线和频谱塑形指标预览。
- 独立混音器：5 段 EQ 曲线、混响、声像、空间宽度和空间延迟。
- 可设置导出时长、采样率和音频格式。
- 支持导出 `WAV 16-bit PCM`、`WAV 32-bit Float`、`AIFF 16-bit PCM`、`AU 16-bit PCM`。
- 启用混音器时导出立体声文件。

## 本地运行

这是静态项目，不需要安装依赖：

```powershell
python -m http.server 5173
```

然后访问：

```text
http://localhost:5173/
```

## 频谱映射

本项目采用人为定义的可解释映射，不声称可见光颜色和声音频谱有物理一一对应关系。

- 色相决定整体频谱斜率和主频带中心。
- 饱和度决定从白噪声到目标频谱的染色强度。
- 中心位置保留较高白/灰噪声权重。
- 黄/绿区域会生成明确的中低频和中频增强，避免低频与高频锚点混合造成长期中频凹陷。
- 频谱预览显示的是当前目标频谱和 EQ 曲线共同作用后的理论倾向。

## 导出说明

导出不会录制实时播放流，而是用当前色盘参数和混音器参数离线生成指定时长的样本，再写入所选音频格式。默认会把峰值归一化到约 `-1 dB`，以减少不同噪声颜色和混音处理之间的响度差异。
