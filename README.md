# 噪声调色盘

[简体中文](README.md) | [English](README.en.md)

一个无依赖的浏览器噪声合成与混音工具。它把“噪声颜色”做成可拖动的色盘，同时提供独立的混音器，用于调整 EQ、混响、声像、空间宽度和导出音频。

## 功能

- 色盘拖动实时控制噪声颜色。
- 中心接近白/灰噪声，越靠外频谱染色越强。
- 内置棕/红、红紫凹陷、黄色中低频、绿色强中频、青色存在感、蓝色明亮、紫色空气感锚点。
- 中英文界面切换，并通过 `localStorage` 记住语言。
- 实时频谱曲线和频谱塑形指标预览。
- 独立混音器：5 段 EQ 曲线、混响、声像、空间宽度和空间延迟。
- 音频噪化：上传源音频后叠加当前调色盘噪声，并让噪声经过当前混音器处理。
- 音频全损：对上传音频进行削波、降采样、降位深、窄带滤波和时间涂抹处理。
- 可设置导出时长、采样率和音频格式。
- 支持导出 `WAV 16-bit PCM`、`WAV 32-bit Float`、`AIFF 16-bit PCM`、`AU 16-bit PCM`。
- 启用混音器时导出立体声文件。

## 频谱映射

本项目采用人为定义的可解释映射，不声称可见光颜色和声音频谱有物理一一对应关系。

- 色相决定整体频谱斜率、主频带中心，以及红紫区间的中频凹陷程度。
- 饱和度决定从白噪声到目标频谱的染色强度。
- 中心位置保留较高白/灰噪声权重。
- 黄色区域保留中低频增强，绿色区域生成更窄、更强的中频峰，两端频率会相对下落。
- 红紫过渡会削弱中频并增强低频和高频，承接红色低频厚度与紫色高频空气感。
- 频谱预览显示的是当前目标频谱和 EQ 曲线共同作用后的理论倾向。

## 导出说明

导出不会录制实时播放流，而是用当前色盘参数和混音器参数离线生成指定时长的样本，再写入所选音频格式。默认会把峰值归一化到约 `-1 dB`，以减少不同噪声颜色和混音处理之间的响度差异。

音频噪化在浏览器本地解码上传文件，保留源音频左右声道，再叠加经过当前调色盘和混音器处理的噪声；输出沿用当前采样率、格式和归一化设置。

音频全损同样在浏览器本地处理上传文件，但它不叠加彩色噪声，而是模拟 meme 常见的低码率、爆麦、电话转载和 bitcrush 损坏链；输出沿用当前采样率、格式和归一化设置。
