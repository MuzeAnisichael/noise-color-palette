# Noise Colour Palette

一个浏览器里的噪声调色盘。用户可以拖动色盘上的点，实时监听对应频谱倾向的噪声，并把当前噪声离线保存成音频文件。

## 功能

- 色盘拖动实时控制噪声颜色。
- 中心接近白/灰噪声，越靠外频谱染色越强。
- 内置棕/红、粉红、绿色、蓝色、紫色噪声锚点。
- 实时频谱曲线和混合比例预览。
- 可设置导出时长、采样率和音频格式。
- 支持导出 `WAV 16-bit PCM`、`WAV 32-bit Float`、`AIFF 16-bit PCM`、`AU 16-bit PCM`。

## 本地运行

这是一个无依赖静态项目，可以直接用本地静态服务器打开：

```powershell
python -m http.server 5173
```

然后访问：

```text
http://localhost:5173/
```

## 频谱映射

本项目采用人为定义的可解释映射，不声称可见光颜色和声音频谱有物理一一对应关系。

- 色相决定靠近哪个噪声锚点。
- 饱和度决定染色强度。
- 中心位置保留较高白/灰噪声权重。
- 频谱预览显示的是当前混合权重对应的理论倾向。

## 导出说明

导出不会录制实时播放流，而是用当前色盘参数离线生成指定时长的样本，再写入所选音频格式。默认会把峰值归一化到约 `-1 dB`，以减少不同颜色噪声之间的响度差异。
