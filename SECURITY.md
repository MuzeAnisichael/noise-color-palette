# Security Policy

[简体中文](#简体中文) | [English](#english)

## 简体中文

如果你发现安全问题，请不要公开创建 issue。请通过 GitHub 私密渠道联系维护者，或在仓库的 Security 页面提交报告。

这个项目是纯前端静态应用，不收集用户数据，也不向远程服务上传音频。导出的音频文件在浏览器本地生成。

## English

If you discover a security issue, please do not open a public issue. Contact the maintainer through a private GitHub channel or submit a report through the repository Security page.

This project is a static frontend app. It does not collect user data or upload audio to remote services. Exported audio files are generated locally in the browser.
